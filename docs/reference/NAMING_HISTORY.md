# Naming History & Alternative Names

This document captures the brainstorming process that led to "Port of Call" and explores alternative names that were considered.

## Final Choice: Port of Call

**Winner!** This name was chosen for its multi-layered meaning and thematic fit.

### Strengths
- ✅ Double pun (literal port + nautical metaphor)
- ✅ Fits Cloudflare's nautical theme
- ✅ Professional yet memorable
- ✅ Tells a story about data transition

### Existing Usage
- Ruby gem: `port-of-call` (Rails port assignment utility)
- Los Alamos Lab: `ports-of-call` (performance portability)
- Maritime industry: Port call logistics software
- **Impact**: Distinct enough - focused on Cloudflare Workers/Sockets API

## Runner-Up: Protocall

A clever tech-focused name that almost made it.

### Strengths
- Punny and professional
- Suggests "protocol" + "call"
- High signal for technical audience

### Conflicts
- Google's archived "Protocall" (Protocol Buffers language)
- Protocall Services (behavioral health company)
- Various small repos on GitHub

## Alternative Names Considered

### Bridge & Gateway Theme
Names focusing on the transition between browser and server:

- **WarpPipe**: Fast data movement between worlds
- **PortJump**: Jumping from port to web
- **ViaWorker**: Clean and professional
- **Bridgehead**: Entry point into network territory

### Nautical & Flow Theme
Aligned with Cloudflare's ecosystem:

- **Jetty**: Structure projecting into water (perfect for browser-to-server bridge)
- **Conduit**: Seamless data flow
- **TidalSocket**: Back-and-forth data movement
- **Outbound**: Focus on egress nature

### Technical & Literal Theme
For clarity about functionality:

- **RawWorker**: Emphasizes raw sockets
- **SocketScript**: Play on Workers scripting
- **WebSSH-Core**: SSH-specific focus
- **SocketLink**: Direct and functional

### 80s/90s Protocol Puns

These lean into retro networking nostalgia:

#### SSH & Telnet
- **ShellShocked**: Browser can't believe it's running a terminal
- **Net-Ill-Set**: Play on Telnet ("sick" in the cool sense)
- **ShellVerve**: Server + Shell with energy
- **WellMet**: Play on Telnet (polite, old-school)

#### TCP/IP & Routing
- **WorkerGopher**: Nod to Gopher protocol ("goes fer' things")
- **ICMP-U-Later**: Internet Control Message Protocol pun
- **SlipStream**: SLIP (Serial Line IP) reference, sounds fast
- **BGP-Easy**: Border Gateway Protocol made simple

#### Hardware & Physical Layer
- **TokenRingLeader**: Managing multiple connections (Token Ring)
- **Ether-Or**: Ethernet play - browser or server? Both!
- **TwistedPairing**: Twisted Pair cables + browser-socket pairing
- **DialUp-Start**: 56k modem vibes for modern upstarts

#### Absolute Nerdiest
- **SevenLayers**: OSI Model reference (handling all 7 layers)
- **SocketToMe**: Classic catchphrase + TCP sockets
- **PromiscuousMode**: Network card joke (intercepts all traffic)
- **SYN-ityCheck**: TCP SYN packet + "sanity check"

## Thematic Branding (Port of Call)

If we lean into the nautical theme, we could use these terms:

- **The Jetty**: Entry-point function
- **The Manifest**: `wrangler.toml` configuration
- **Anchoring**: Persistent connection handling
- **Harbor**: Collection of available services
- **Voyage**: Active TCP session
- **Docking**: Connection establishment
- **Cargo**: Data payload
