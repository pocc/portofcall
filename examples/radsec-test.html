Note: RADSEC (RADIUS over TLS, RFC 6614) provides secure AAA (Authentication, Authorization, Accounting) over TLS-encrypted connections. This tester sends Access-Request packets to a RADSEC server on port 2083.

Connection Settings

Authentication Credentials

Security: Credentials are encrypted by TLS during transmission. RADSEC provides stronger security than traditional RADIUS (which uses MD5 obfuscation).

NAS Information (Optional)

NAS (Network Access Server): The device requesting authentication (e.g., WiFi access point, VPN gateway). These fields are optional but commonly used in enterprise deployments.
