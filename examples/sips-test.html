Note: SIPS (SIP Secure, RFC 3261/5630) is SIP over TLS. It provides encrypted signaling for VoIP calls, video conferences, and instant messaging. This tester sends SIPS OPTIONS or REGISTER requests to a SIP server over TLS (port 5061).

Connection Settings

SIP URIs

SIPS URI Format: sips:user@domain
Example: sips:alice@example.com
Note: Use "sips:" scheme for encrypted connections

Authentication (for REGISTER)

Note: Most SIP servers require authentication for REGISTER. This implementation sends the initial request without credentials. If the server responds with 401 Unauthorized, you would need to implement digest authentication (follow-up request with Authorization header).
